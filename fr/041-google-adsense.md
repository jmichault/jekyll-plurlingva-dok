# pré-requis
* un compte google
* une adresse physique
* un numéro de téléphone


# Activer
* Aller sur le site (_Google AdSense)[https://www.google.com/adsense/]
* S'identifier
* Renseigner le profil
* Renseigner les paramètres des annonces
* connecter le site
* cliquer sur obtenir le code, et noter le paramètre _client_
* renseigner ga_adsense dans votre fichier \_config.yml avec le code noté au-dessus
