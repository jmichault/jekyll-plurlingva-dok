あなたの訪問者があなたのページの1つにコメントを追加することを可能にするために必要です（°1°1

# 現場で
 * GitHubのユーザー名で _gh\_uzanto_ を _gh\_uzanto_ で見つけます。



# ポスターやページのために
 *  _«issue»_ サイトに対応するリポジトリに _«issue»_ 、その数に注目してください。 （°3°3



 * ポスターやページヘッダー _YAML_ に追加する：（°3°3  



```
komentoj_id: #
```
(早期)の数で _#_ 交換 _#_
