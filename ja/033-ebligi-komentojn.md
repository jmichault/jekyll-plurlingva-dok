あなたの訪問者があなたのページの1つにコメントを追加することを可能にするために必要な必要があります：

# 現場で
 * GitHubのユーザー名で _gh\_uzanto_ を _gh\_uzanto_ で見つけます。



# ポスターやページのために
 * サイトに対応するリポジトリに _«issue»_ を作成し、その数に注意してください。 



 * ポスターやページヘッダーに追加する _YAML_ ：  



```
komentoj_id: #
```
(早期)の数で _#_ 交換 _#_
